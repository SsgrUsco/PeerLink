Uso general

No puedo iniciar sesion

Verifica correo, contrasena y que el navegador permita el certificado local si trabajas en entorno de desarrollo.

Como solicito una tutoria

Desde Tutorias puedes usar Solicitar para proponer fecha y tutor, o Aceptar para reservar una tutoria ya publicada.

Tutores

Como publico una tutoria

Desde Crear tutoria filtra la materia, selecciona la coincidencia y asigna fecha y hora.

Que puedo hacer como admin

Puedes crear materias, asignar tutores, administrar usuarios y abrir Swagger con permisos protegidos.

Si sigues bloqueada, comparte una captura y el mensaje de error con soporte o con el administrador.
