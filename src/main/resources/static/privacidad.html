Politica de privacidad

PeerLink utiliza datos de cuenta, rol, materias y reservas unicamente para gestionar autenticacion, tutorias y paneles de trabajo.

La informacion almacenada se limita a los datos necesarios para operar la plataforma y mostrar a cada persona el contenido correspondiente a su rol.

Las credenciales se gestionan de forma segura y la aplicacion no debe compartir datos con terceros no autorizados dentro del alcance actual del proyecto.
